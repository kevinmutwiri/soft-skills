# The World Wide Web

> [home](./README.md)

The web was invented by Sir Tim Berners-Lee in the late 1980s while working at CERN, the European Organisation for Nuclear Research. His vision was to create a system that allows researchers to easily share and access scientific information. As you watch [this video](https://www.youtube.com/watch?v=lYUMqsqNAdM) on the internet’s history, make sure you can answer the following questions:

- What is the World Wide Web (WWW)?
- What is the difference between WWW and the Internet?
- Where does the WWW come from?
